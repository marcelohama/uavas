Installation:
=============

copy all the files included in this package in the FMS model folder :
c:\Program Files\FMS\Model

The files required for the QRO model are :
qrov21.x
qrov21.wav
qrov21_prv.jpg
qrotext.bmp
qromtxt.bmp
qrov21.par

Then launch the FMS simulator software and load the model (qrov21.par) with the key F2 and Enjoy....
